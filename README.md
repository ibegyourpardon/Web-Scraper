# WEB SCRAPER  
 
Python Web Scraper made using Flask framework, MongoDB and APScheduler   

**To use**

- run `python app.py`
- Navigate to `localhost:5000`

**To install**

- clone repo
 
**To run tests**

- run `python -m unittest -v test_file.Testing`
